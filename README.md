## 日本語

### 「半透明」が綺麗に抜けない悩みを『AaronAlpha』で解決する方法
👉 **[アプリを起動する (Launch App)](https://aaronalpha.aaronpoint.workers.dev/)**

### 💻 動作環境
- **PCのみ対応** (タブレットやスマートフォンは画面サイズやUIの都合上、動作対象外です)

### 🌟 概要
半透明やグロー効果（発光）を含んだ画像を**綺麗に抜きたい**と思ったことはありませんか？
AI背景除去を使っても、依然として半透明部分の抽出は困難です。

このツールは、同じ被写体の黒/白背景画像から**半透明を綺麗に抜く**透過png出力ツールです。

ブラウザ（ローカル）で完結し、画像がサーバに送信されることはありませんので安心してご利用いただけます。

### 🚀 主な機能
1. **🔰 基本モード (初心者向け)** 黒背景と白背景の画像をドラッグ＆ドロップして「PNG出力」を押すだけの簡単操作。ズレの自動補正も行います。
2. **🗑️ ゴミ取りモード (中級者向け)** AI生成画像などに残る微細な背景ノイズを削り落とし、完璧な透明に追い込むためのモード。投げ縄ツールによる編集やガベージマットにも対応。
3. **🟦 ソリッドモード (上級者向け)** 本来完全な不透明であるべき部分（ソリッド）に残る「透け」を、完全な不透明へ追い込みます。

詳しくはツール内の「使い方」ボタンをご覧ください。

### 💡 ヒント
※このツール自体に画像生成機能はありません。
「使い方」ボタンで表示されるヘルプに、サンプル画像も用意してあります。
まずはこのサンプルでお試し下さい。

**入力画像を生成するコツ**
「背景を黒単色（#000000）にして下さい。位置や大きさなどは一切動かさず忠実に再現して下さい」
さらに「xxの部分は半透明」「xxにはグロー効果が掛かっている」などと補足すると精度が上がったりします。

## English

### How to perfectly extract semi-transparent subjects with AaronAlpha
👉 **[Launch Application (アプリを起動する)](https://aaronalpha.aaronpoint.workers.dev/)**

### 💻 Operating Environment
- **PC Only** (Tablets and smartphones are not supported due to screen size and UI constraints)

### 🌟 Overview
Have you ever wanted to **cleanly extract** images that contain semi-transparency or glow effects?
Even with AI background removal tools, extracting semi-transparent areas remains a difficult task.

This tool is a transparent PNG extractor that **cleanly extracts semi-transparency** using black and white background images of the exact same subject.

Your processing is completed entirely in your browser (local). Your images are never uploaded to any server, so you can use it with complete peace of mind.

### 🚀 Key Features
1. **🔰 Basic Mode (Beginners)** A simple operation: just drag and drop the black and white background images and click "Export PNG". It also features automatic alignment correction.
2. **🗑️ Garbage Removal Mode (Intermediate)** A mode to eliminate fine background noise leftover in AI-generated images, pushing the background to perfect transparency. It supports editing with a lasso tool and garbage matting.
3. **🟦 Solid Mode (Advanced)** Pushes the "see-through" left in parts that should originally be completely opaque (solid) back to complete opacity.

For more details, please see the "Help" (使い方) section within the tool.

### 💡 Tips
*This tool itself does not generate images.*
Sample images are available in the help section displayed via the "Help" button.
Please feel free to try it out using these samples first.

**Tips for generating input images:**
"Please make the background pure black (#000000). Please reproduce the subject faithfully without changing its position or size at all."
Furthermore, adding supplements like "The xx part is semi-transparent" or "The xx has a glow effect" can sometimes improve the consistency of AI generation.
