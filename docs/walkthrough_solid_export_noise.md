# ソリッドモード書き出し時のノイズ除去完了報告

## 現象
ソリッドモードでの書き出し（透過PNG）の際、透明であるべき箇所にノイズ（壊れたRGB値）が混入する。

## 原因
finalize_solid_mode() において、RGBバッファのクリアが不完全なまま合成が行われていた。

## 修正内容
- 書き出し用バッファ（solid_export_buffer）の構築プロセスの冒頭で、全ピクセルを (0, 0, 0, 0) でクリア。
- アルファが 0 のピクセルに対しては、RGB計算をバイパスして純粋な透明黒を保証。
- プレビュー表示用と書き出し用のバッファ処理を分離し、整合性を高めた。
