# パフォーマンスおよびプレビュー表示の不具合修正報告

## 修正項目
1. **位置調整（Lasso Alignment）のレスポンス改善**:
    - ビューポート計算が未定義だった箇所を修正し、等倍表示（100%）時でも移動が重くならないよう最適化。
2. **ズーム機能の安定化**:
    - fitToScreen 後の拡大率計算の不整合を解消。
3. **Rust側の高速パス適用**:
    - update_alignment_alpha_only において、不要なバッファ更新をスキップし、描画負荷を最小限に抑制。

## 結果
以前発生していた「位置調整中のフリーズ」や「移動の極端な遅延」が解消され、滑らかな操作が可能となりました。
