# Aaron Alpha 操作マニュアル (v2.0)

このドキュメントでは、Aaron Alphaの全機能と使い方のコツを解説します。
Aaron Alphaは、3つのステップ（モード）を使い分けることで、究極の透過画像を作成できるツールです。

---

## 1. 基本モード 【初級者：まずはここから】
2枚の画像（黒背景・白背景）を合成し、背景を透過させる基本的なステップです。

- **入力画像の用意**:
    - 全く同じポーズで「黒い布（または背景）」と「白い布」の前で撮影した2枚の画像を使用します。
    - ドラッグ＆ドロップで各ゾーンに放り込むだけで、自動で位置ズレ（Offset）が計算されます。
- **合成アルゴリズム (Logic)**:
    - **平均法 (Avg)**: 最も一般的で自然な結果が得られます。
    - **Max法 / Lum法**: 画像の輝度や色味によって微調整したい場合に使用します。
- **補正値の確認**:
    - 自動計算されたOffset値が右下に表示されます。基本的には触れる必要はありません。
- **PNG出力**:
    - 納得がいったらボタンを押して保存。初級者はここで満足してもOKです。

---

## 2. ゴミ取りモード 【中級者：そのこだわりがプロへの道】
基本モードでは消しきれなかった「背景のシワ」や「ノイズ」を徹底的に除去するステップです。

- **位置調整 (📍 位置調整)**:
    - 2枚の画像に微細なズレがある場合、矢印キーで1ピクセル単位の調整が可能です。
    - **[A]キー** を押している間は背景が切り替わり、ズレを視覚的に確認（クイックプレビュー）できます。
- **ノイズ閾値 (Threshold)**:
    - スライダーを動かすことで、背景に残った細かなゴミを消し込みます。
- **範囲除外設定 (📐 除外範囲設定)**:
    - 画面の端にある機材や余計な映り込みを、マウスドラッグで囲ってバッサリとカットします。
- **投げ縄ツール (Lasso)**:
    - **[Alt]キー** を押しながらマウスで囲むことで、自由な形で透過範囲を指定できます。
    - 失敗しても **[Ctrl + Z]** でUndo可能です。
- **AA (アンチエイリアス)**:
    - 境界線を滑らかにするかどうかを選択できます。

---

## 3. ソリッドモード 【上級者：究極のクオリティを求める職人へ】
透けてほしくない部分（不透明度100%にすべき場所）を完璧に保護する、真の職人のための最終ステップです。

- **不透明閾値 (Solid Level)**:
    - AIによる解析とレベル補正を組み合わせ、中身が透けない「ソリッド（固体）」な領域を抽出します。
- **可視化表示**:
    - **青い領域**: システムが自動で抽出したソリッド範囲。
    - **緑の領域**: あなたが手動で描き足した、あるいは充填した「信頼の証」となる範囲。
- **充填ショット (🎯 充填)**:
    - 中心部を指定すると、周囲のソリッド領域を爆速で埋め尽くします。
    - 完了するとボタンがパッと緑色にフラッシュします。
- **自作マットのロード**:
    - 自分で作成した白黒画像を読み込んで、ソリッド範囲として利用することも可能です。
- **プレビュー (👁️ プレビュー)**:
    - 実際の合成結果とソリッド範囲を重ねて確認。納得いくまで調整を。

---

## 共通操作 & ショートカット
職人なら指に覚え込ませたい、便利な操作一覧です。

- **キャンバス操作**:
    - **マウスドラッグ**: 画像の移動（パン）
    - **ホイール**: ズームイン / ズームアウト
- **ショートカットキー**:
    - **[A]**: 背景色と市松模様を即座に切り替え（確認用）
    - **[Ctrl + Z]**: 元に戻す（Undo） ※設定から回数を3〜10回で変更可能
    - **[Z] (押しっぱなし)**: 比較モード中でも強制的に投げ縄ツールを使用可能に
    - **[Alt] (押しっぱなし)**: 投げ縄（Lasso）モード開始
- **設定 (歯車アイコン)**:
    - **言語**: 日本語 / 英語の切り替え
    - **速度優先モード**: 位置調整時のプレビューを爆速にします。
    - **投縄Undo**: 自分の腕に合わせてUndo回数を調整（3〜10回）。
