# Aaron Alpha 使い方マニュアル (v1.0)

このドキュメントはアプリケーション内の「使い方」や各機能のツールチップ（詳細説明）に記載されているヘルプ文言をまとめたものです。

---

## 🚀 ツール紹介

半透明やグロー効果（発光）を含んだ画像を、**綺麗に切り抜きたい**と思ったことはありませんか？
AI背景除去を使っても、依然として半透明部分の抽出は困難です。
NanobananaやMidjourneyは透過PNGを生成できません。

このツールは、同じ被写体の黒/白背景画像から、**半透明を綺麗に抜くPNG出力ツール**です。
> **※ 注意**：このツール自体に入力画像を生成する機能はありません。

ブラウザ（ローカル）で完結し、画像がサーバに送信されることはありません。

### ※入力画像を生成するコツ
「背景を黒単色（`#000000`）にして下さい。位置や大きさなどは動かさず忠実に再現して下さい」
さらに「xxの部分は半透明」「xxにはグロー効果が掛かっている」などと補足すると精度が上がります。
AIは気まぐれで、位置や色味、縮尺が異なる場合があります。ご注意下さい。

## 🔰 基本モード（初心者向け）

### 手順概要
1. 黒背景・白背景の2枚の画像をD&D（ドラッグ＆ドロップ）。
2. 「PNG出力」ボタンを押す。以上。

さらに細かく調整したい場合は **ゴミ取りモードへ**！

### キー・マウス操作一覧
- **ホイール**: 拡大縮小
- **[Space] ＋ 左ドラッグ** (または**ホイールクリック**) : スクロール
- **[Space] ＋ 右クリック** (または**ホイールダブルクリック**) : 画面フィット
- **[A] キー** : 押している間は背景設定が一時的に反転します。

### 設定項目の解説
- **入力画像**: 同サイズの黒/白背景画像をD&Dして下さい。ズレがある場合は自動で補正します。
- **Logic**: 
  - `平均法`: 一般的で自然な結果が得られます。
  - `Max法` / `Lum法`: 輝度や色味が少し変化します。
- **補正値**: 自動で補正された値を表示します。
- **比較**: スライダーで元画像と比較できます。
- **PNG出力**: 透過PNGを出力します。

**【ソリッド＆半透明保護カーブ】**
- **定着ライン (S)**: この閾値以上のアルファが完全不透明に引き上げられます。
- **保護パワー (P)**: 値が高いほど半透明部分が強く保護されます。

## 🗑️ ゴミ取りモード（中級者向け）

### ゴミ取りモードとは
一見綺麗に見えるAI画像も、背景には微細なノイズが大量に残っています。
ここではノイズ閾値を調整し**背景ノイズを透明に追い込みます。**
数値を上げすぎると、必要なディテールまで削れるため注意してください。

除去した結果は基本モードに反映されます。調整が終わったら、基本モードに戻って出力してください。
さらに不透明部分を補正したい場合は **ソリッドモードへ**！

### キー・マウス操作一覧（追加分）
- **投縄ツール**
  - **左ドラッグ**: アルファを削る（マスキング）
  - **右ドラッグ**: 元に戻す
  - **[Alt] ＋ クリック**: 多角形で囲む
  - **[Z] キー**: 押している間は、比較スライダー操作中も投げ縄機能を強制的にオンにします。
  - **[Ctrl] + [Z]**: Undo（やり直し）

### 設定項目の解説
- **ノイズ閾値**: 透明に近いノイズを除去します。
- **位置調整**: 手動でズレを調整します。画像がズレていると不透明部分に暗い場所が沢山できます。なるべく全体が白くなるように調整して下さい。
- **除外範囲設定**: 選んだ範囲外が全て透明になります（ガベージマット機能）。上下左右の数値入力で微調整も可能です。
- **表示切り替え**:
  - `2値表示`: 1ドットでも存在している場所を白で視覚化します。
  - `アルファ表示`: アルファの状態を視覚化します。
  - `プレビュー`: アルファを適用した画像を表示します。
- **アンチエイリアス**: 投縄の境界を滑らかにします。
- **ソリッド焼付**: ソリッドモードで編集したソリッド部分をアルファに焼き付けます（通常利用する必要はありません）。
- **PNG出力**: 外部ツールでのマットや下敷き用に、白黒マットやアルファマットを出力します。

## 🟦 ソリッドモード（上級者向け）

### ソリッドモードとは...
黒白比較の抽出では、本来ソリッド（完全な不透明）であるべき部分にも「透過情報」が残ってしまうことがあります。
不透明度閾値を調整して**ソリッド部分を完全な不透明に追い込みます。**

ただし、下げすぎると半透明やグロー、アンチエイリアスの部分まで不透明にしてしまうので、やり過ぎは禁物です。そもそも半透明が主体の場合はココの編集自体が不要です。

「充填」ボタンで塗り残しを自動補正し、細かい部分は「投げ縄」で修正してください。
> **💡 ヒント**: 白（ソリッド）を編集するので「可視化オン」での作業がオススメです。慣れてきたらプレビューと比較をオンにして、元画像と比較しながら気になる所だけ修正すると効率的です！

### 最後に
もっと詳しく知りたい方は、左上の **鹿のアイコン** からどうぞ。

### キー・マウス操作一覧（追加分）
- **投縄ツール**
  - **左ドラッグ**: 不透明にする
  - **右ドラッグ**: 元に戻す（透過状態に戻す）
  - ※ その他の操作はゴミ取りモードと同じです。

### 設定項目の解説
- **不透明度閾値**: 不透明に近い場所を完全な不透明に追い込みます。ある程度「青」で埋まるまで調整して下さい。
- **自作マットをロード**: 上級者向け。任意で自作のマット画像をロードできます。
- **可視化**: 完全不透明部分を「緑」で可視化します。
- **充填**: 漏れた部分を自動で埋めます（青が緑に変わります）。穴埋めロジックには「探索距離」が影響し、あまり大きくするとソリッド外まで塗り潰してしまうので注意が必要です。
- **アンチエイリアス幅** & **閾値**: （隠れ機能有効時）アンチエイリアスとして確保する輪郭の幅とアルファ値を設定します。`[S]キー`でカーソル直下のアルファ値を自動設定できます。

## ⚙️ 設定画面について

アプリケーション全体の動作設定や隠し機能の有効化を行います。
設定内容はブラウザ（ローカル）に保存されます。

- **言語**: 日本語・英語を切り替えます。
- **手動位置調整速度優先**: 低スペックPC用。手動位置補正時、低画質で描画スピードを優先します。
- **隠れ機能**: 実験的な機能メニューを有効化します。
- **位置自動補正**: 画像D&D時に自動で位置ズレを計算・補正します。
- **投縄Undo**: 投げ縄操作のUndo回数を設定します（3〜10回）。
- **全ての設定を初期化**: 初期状態に戻します（認証キー等は残ります）。
